# 파이썬 웹 청첩장 (강아지 & 고양이)

이 프로젝트는 플라스크(Flask)로 만든 간단한 웹 청첩장입니다.

- **신랑**: 강아지  
- **신부**: 고양이  
- **신랑 측 부모님**: 개똥이, 말숙이  
- **신부 측 부모님**: 초롱이, 두리  
- **예식장 위치**: 서울특별시 동대문구 왕산로 200 (구글 지도 임베드)

## 실행 방법 (로컬)

1. 파이썬 가상환경(선택 사항)이 있다면 활성화합니다.
2. 필요한 패키지 설치:

```bash
pip install -r requirements.txt
```

3. 서버 실행:

```bash
python app.py
```

4. 브라우저에서 `http://localhost:5000` 에 접속하면 웹 청첩장을 볼 수 있습니다.

## 고양이 & 강아지 결혼 사진 넣기

1. 마음에 드는 **고양이와 강아지가 결혼하는 느낌의 사진**을 준비합니다.
2. 파일 이름을 `cat_dog_wedding.jpg` 로 저장합니다.
3. 이 프로젝트의 다음 경로에 파일을 넣습니다.

```text
static/images/cat_dog_wedding.jpg
```

이미지를 넣으면 메인 페이지에 사진이 자동으로 표시됩니다.

## Docker / docker-compose 로 실행 (선택)

1. 도커 이미지 빌드 및 실행:

```bash
docker-compose up --build
```

2. 브라우저에서 `http://localhost:5000` 에 접속합니다.
